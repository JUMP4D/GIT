Installation de Git

Pour commencer à utiliser Git sur votre système, vous devez d'abord l'installer.

Windows

  • Téléchargez l'installateur Git pour Windows à partir du site officiel de Git : Télécharger Git
  • Exécutez l'installateur téléchargé et suivez les instructions à l'écran pour installer Git sur votre système.

macOS

  • Vous pouvez installer Git sur macOS en utilisant Homebrew, un gestionnaire de paquets pour macOS. Ouvrez Terminal et exécutez la commande suivante : brew install git

Linux

  • Sur la plupart des distributions Linux, Git est généralement disponible dans les dépôts de paquets par défaut. Vous pouvez l'installer en utilisant le gestionnaire de paquets de votre distribution. Par exemple, sur Ubuntu, vous pouvez utiliser apt : sudo apt-get install git

Une fois Git installé sur votre système, vous pouvez vérifier si l'installation a réussi en ouvrant un terminal et en exécutant la commande : git --version

Vous devriez voir s'afficher la version de Git installée sur votre système. Félicitations, vous êtes maintenant prêt à commencer à utiliser Git pour gérer vos projets de développement !
