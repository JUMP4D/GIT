Commandes de base de Git

Git offre un ensemble de commandes essentielles pour gérer efficacement vos projets de développement. Voici quelques-unes des commandes de base que vous utiliserez fréquemment :

git init

Initialise un nouveau dépôt Git dans le répertoire courant.

git add

Ajoute des modifications de fichiers spécifiques à l'index pour le prochain commit.

git commit -m "message"

Enregistre les modifications ajoutées à l'index dans un nouveau commit avec un message descriptif.

git status

Affiche l'état actuel du répertoire de travail et de l'index.

git log

Affiche l'historique des commits du projet.

git branch

Affiche la liste des branches locales et indique celle sur laquelle vous êtes actuellement.

git checkout

Basculer sur une autre branche.

git merge

Fusionne une branche spécifiée dans la branche actuelle.

git pull

Récupère les modifications depuis un dépôt distant dans votre branche locale.

git push

Envoie vos modifications locales vers un dépôt distant.
